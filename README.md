# CTF Challenge Archive

Archive of the BB3 [bb3.tech](https://bb3.tech).  
Each folder includes:
- `response.json` — raw challenge data
- `description.txt` — clean, readable summary (auto-generated)

Organized by category: **Cryptography**, **Web**, **Reverse**, etc.  
Metadata includes name, ID, points, solves, description, file links, and hints.

---

## How to Contribute a Solution

Solved a challenge? Help others learn — submit a **pull request**!

1. Add a file: `solution.md` inside the challenge folder  
2. Use this exact format:

# Flag
DJSISACA{your_flag_here}

# Explanation
Step-by-step writeup:
- Tools or scripts used
- Key observations
- Full reasoning and steps to solve

# Screenshots (optional)
Include images or terminal output if they clarify the process.


### Folder Example
```
Web/
└── 15_SQL_Injection_Basics/
    ├── response.json
    ├── description.txt
    └── solution.md

```
Clear, beginner-friendly writeups are prioritized for merge.
